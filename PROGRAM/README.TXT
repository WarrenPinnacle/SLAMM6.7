SLAMM Open-Source Code Distribution, April 2017 or later
Code Produces SLAMM 6.7 beta-- Build 6.7.0236 or later
Code is object oriented Pascal written with Borland XE3 development platform

This code is licensed under the Common Public License (CPL), see SLAMM_License.txt

------------------------------------------------------------------------------------

To compile this source code as-is, the following steps must be taken:

1. Install Borland Delphi XE3 for WIN32.   

2. Unzip this zip file retaining the directory structure.  This will be your main program directory.

3. Select File, Open Project, and Load SLAMM6.dproj file  Executable may be produced by selecting 
  "Build" from the Project menu.
